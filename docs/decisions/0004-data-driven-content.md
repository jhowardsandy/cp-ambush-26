# ADR 0004: Data-Driven Content

Status: Accepted.

Scenarios, units, abilities, terrain, effects, and objectives will be defined as data wherever practical.

Implementation update, 2026-07-20: portable scenario JSON, terrain, named map areas, objectives, unit archetypes, faction roster packages, effects, and direct-attack profiles now have initial schemas and validators. Content version is recorded in scenario/replay inputs. Capability binding, equipment, progression, import tooling, and migrations remain deferred until their rules exist.
