# Iron Timeline — Working Title and Description

## Title status

**Iron Timeline** is the current working title for the military tactical game. It is provisional and does not represent a trademark, company name, or final branding decision.

The title reflects the game's defining mechanic: individual orders resolve on one shared tactical timeline.

Other candidate working titles retained for later review:

- Crossfire Command
- Field Order
- Tactical Horizon
- Ambush Line
- Command at Dusk

## Family-facing description

**Iron Timeline** is a new tactical strategy game in development by **Justin Howard**.

Inspired by the tension and thoughtful planning of classic computer wargames, *Iron Timeline* is built around commanding individual units across a shared battlefield timeline. Every move, attack, heal, and tactical decision unfolds according to clear, replayable rules—so strategy matters as much as quick thinking.

The game will grow from small squad encounters into larger maps with terrain, buildings, cover, hills, vehicles, and distinct unit abilities. It is being created as a modern, original tribute to the kind of turn-based military strategy games that made early computer gaming so memorable.

Personal note for family sharing: Justin is building it in part for the nostalgia and tactical joy those early computer strategy games created.

## Brand guardrail

The game will remain original in its name, art, maps, rules expression, content, and presentation. It may evoke the design pleasures of classic tactical computer games without using their protected branding or content.
