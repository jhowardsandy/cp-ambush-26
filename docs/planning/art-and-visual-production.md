# Art and Visual Production Plan

Status: proposed production plan. It does not define a final visual style yet.

## Timing

Do the first playable PvE test as a clear graybox encounter before a final art pass. The test needs readable units, terrain, range/overwatch feedback, posture, event explanations, and a small original scenario; it does not need finished characters, landscapes, or branding. Human feedback from that proof should inform the visual brief.

## Three-stage art plan

1. **Readable prototype kit:** faction colors, distinct unit silhouettes, terrain tiles for open ground/brush/cover/blocker, simple status icons, and core action feedback. Continue using primitives where they are clearer than unfinished art.
2. **Vertical-slice kit:** establish an original art-direction board; create a small modular environment kit (ground, roads, buildings, walls, brush, trees, rocks); four-to-six unit archetype tokens; UI iconography; and minimal state animations: idle, move, crouch, prone, fire, hit, incapacitated, and overwatch/aim.
3. **Production pipeline:** final original model/sprite standards, materials/lighting, map-prop library, VFX/SFX, accessibility variants, logo/title treatment, store assets, and performance/asset-import rules. Brand/name clearance happens before public release.

## Tooling policy

Unity Terrain tools may be useful later for broad 3D landscapes, height, texture painting, vegetation, and neighboring terrain tiles. The early tactical maps should favor authored modular grid tiles and props because their gameplay semantics (cover, concealment, blockers, elevation) must map directly to scenario data.

Unity AI and other generative tools can accelerate concept exploration and placeholder production. Treat generated assets as drafts: review originality/license terms, normalize style, optimize/import them deliberately, and never let an image, mesh, animation, or prompt decide gameplay rules. Maintain an asset ledger with source/tool, prompt/reference provenance, license, revision, and in-game use.

## First art deliverables after graybox PvE

- One visual-direction board with three original reference directions.
- A terrain readability palette tied to cover/concealment/blocking semantics.
- Blue/red placeholder unit families with role-distinct silhouettes.
- A small icon set for AP, health, posture, overwatch, and objective state.
- A title/logo exploration brief for *Iron Timeline* and alternate working names.
