# Scenario Fixtures

These portable JSON scenarios are content fixtures, not Unity scenes. They provide versioned inputs for core validation, replay, scenario acceptance tests, future AI planning, and presentation.

- `movement-sandbox-01.json`: current small visual sandbox data.
- `iron-timeline-squad-skirmish-01.json`: first larger 16×12, four-versus-four roster fixture with explicit AP budgets, faction/archetype content, named map areas, and an eliminate-all-opponents objective. It now drives the repeatable four-round graybox PvE acceptance test. Search/rescue and reinforcement areas are present as content references only; their behaviors remain future rules.
